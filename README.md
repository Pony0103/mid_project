# 2024 ATP Finals(Singles)
GitHub Pgaes:https://pony0103.github.io/mid_project/
## 簡介
這是一個2024 ATP Finals的賽後結果網頁，方便使用這用最直觀方式查找結果，設有日期選單、球員資本資訊、賽事精華等等...。

## 功能
- 首頁為年終賽簡介
- 左邊的選單列表可進入賽程階段、球員資料
- 賽程階段可瀏覽分組名單、對戰結果
- 球員資料可以瀏覽基本的球員資料

## 使用技術
- 使用HTML建構不同頁面的內容
- 使用React組件動態互動功能
- 使用CSS美化網頁
